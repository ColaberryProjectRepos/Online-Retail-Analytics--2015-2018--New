 Create DAX measures

Steps:

  1. Select the Online Retail analytics Table under the fields pane by clicking on it 
  2. From Power BI desktop ribbon, Home tab click on New measure 
  3. The formula bar appears along the top of the report canvas. 
  4.  Fill in all the DAX formulas provided below, one at a time, and when you finish Press Enter or select Commit (checkmark icon) in the formula bar.  
  5. After creating each measure choose the necessary format (currency) and 2 decimal for each value (measure)
  6. Save
DAX Formulas:
 
Total Sales = SUM(Online Retail Analytics[Sales])
 
Sales Per Month = DIVIDE([Total Sales],DISTINCTCOUNT('Calendar'[Month Name Short]),0)+0
 
Sales YOY = VAR PYSales = CALCULATE([AVG Sales], DATEADD('Calendar'[Date],-1,YEAR)) RETURN IF(ISBLANK(PYSales),0,DIVIDE(([AVG Sales]-PYSales),PYSales))
True Arrow GPM YOY% = SWITCH(TRUE(), [GPM YoY%]<0, UNICHAR(9662),[GPM YoY%]=0, UNICHAR(9666), UNICHAR(9652))
 
 
Discount% = DIVIDE(SUM('ONLINE RETAIL ANALYTICS'[Discount]), SUM('ONLINE RETAIL ANALYTICS'[Sales]), 0)
GPM = 
IF(
    ISBLANK(SUM('ONLINE RETAIL ANALYTICS'[Sales])) || SUM('ONLINE RETAIL ANALYTICS'[Sales]) = 0,
    BLANK(),
    DIVIDE(SUM('ONLINE RETAIL ANALYTICS'[Profit]), SUM('ONLINE RETAIL ANALYTICS'[Sales]))
)
Arrow YOY% = SWITCH(TRUE(), [Sales YOY] <0, UNICHAR(9662),[Sales YOY] =0, UNICHAR(9666), UNICHAR(9652))
 
.Arrow GPM YOY% = SWITCH(TRUE(), [GPM YoY%] <0, UNICHAR(9662),[GPM YoY%] =0, UNICHAR(9666), UNICHAR(9652))
 
GPM YoY% =
VARGPM_CurrentYear= [GPM]
VARGPM_PreviousYear=
    CALCULATE(
        [GPM],
        FILTER(
            ALL('ONLINE RETAIL ANALYTICS'),
            'ONLINE RETAIL ANALYTICS'[Year]= MAX('Calendar'[Year]) - 1
        )
    )
RETURN
IF(
    ISBLANK(GPM_CurrentYear) || ISBLANK(GPM_PreviousYear),
    BLANK(),
    DIVIDE(GPM_CurrentYear- GPM_PreviousYear, GPM_PreviousYear)
)
 
Formats:
  1. Total Sales = Currency, Decimals = 2
  2. Sales per Month = Currency, Decimals = 0
  3. Sales YOY = Percentage, Decimals = 2
  4. Arrow YOY% = Text
  5. Arrow YOY GPM % = Text
  6. discount%= Percentage, Decimals=2
  7. GPM = Percentage, Decimals = 0
  8. True Arrow GPM YOY%=Text
